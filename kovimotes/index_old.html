koviMotes Install Instructions

Note: This requires having BetterDiscord installed.

  1. Right-Click the link to this plugin and Save Link As..
  2. Copy the downloaded koviMotes.plugin.js to %AppData%/BetterDiscord/Plugins.
    3. (Overwrite or delete the old KoviMotes.plugin.js/koviMotes.plugin.js if you still have it or downloaded it from the previous website)
    (If you see CustomMotes.plugin.js or CustomEmotesModified.plugin.js in this folder, go ahead and delete them.)
  3. Reload Discord (Ctrl+R)
  4. Once Discord and BetterDiscord have fully loaded, open Settings (gear below Channel List)
  5. Open the BetterDiscord settings tab on the lower-left, and then the Plugins tab on the top.
  6. Ensure that koviMotes v2017.08.14.2145 by kovibear is on the list, and check "Enabled"
  7. Close out of Settings and the plugin should now be enabled and working. If at any time you wish to update the emotes list, simply Reload Discord (Ctrl+R) and the plugin should fetch the updated list.

This plugin has specifically been built to pull the code and emote list remotely so will no longer need manually downloaded updates.
