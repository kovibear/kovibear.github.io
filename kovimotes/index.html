KoviMotes Install Instructions

  1. Fully shut down Discord (click Quit on the right-click menu in the system tray)..
  2. Click or Right-Click the link to this exe and Save Link As..
  3. Note: your browser may complain about the file, but don't worry about that..
  4. Run the exe to install..
  5. Note: Windows may complain about the file as well, but you can click "More info.." and "Run anyway" to override..
  6. You can now restart Discord..
  7. All done! KoviMotes will now run automatically and persist through Discord restarts!

Manual Instructions

For those uncomfortable using an exe..
  1. Fully shut down Discord (click Quit on the right-click menu in the system tray)..
  2. Click or Right-Click the link to this file and Save Link As..
  3. Note that .asar files are node package files and can opened with any text editor..
  4. Open %AppData%\discord\0.0.305\modules\discord_desktop_core in your file explorer..
  5. Copy/rename the existing core.asar to core.asar.bak (or whatever name you like) as a backup..
  6. Copy the core.asar file you downloaded above into its place..
  7. You can now restart Discord..
  8. All done! KoviMotes will now run automatically and persist through Discord restarts!

Uninstall Instructions

  1. Fully shut down Discord (click Quit on the right-click menu in the system tray)..
  2. Open %AppData%\discord\0.0.305\modules\discord_desktop_core in your file explorer..
  3. Ensure that you have a core.asar.bak (or whatever name you chose) file..
  4. Delete the current core.asar file..
  5. Copy/rename the backup file to core.asar..
  6. You can now restart Discord..
